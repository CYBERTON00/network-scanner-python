# 🌐 Network Scanner (Python)

A lightweight network scanning tool that discovers devices connected to a local network.

## 🚀 Features

* Detect active devices on network
* Shows IP & MAC address
* Fast ARP scanning
* Beginner-friendly cybersecurity project

## 📦 Installation

```bash
git clone https://github.com/CYBERTON00/password-strength-checker.git
cd network-scanner
pip install -r requirements.txt
```

## ▶️ Usage

```bash
python scanner.py
```

## 🧪 Example

```
Enter IP range: 192.168.1.1/24

IP Address        MAC Address
----------------------------------------
192.168.1.1       aa:bb:cc:dd:ee:ff
192.168.1.5       11:22:33:44:55:66
```

## ⚠️ Disclaimer

Use this tool only on networks you own or have permission to scan. Ethical use is strongly encouraged.
